## Requirements:
### SDL2

### Works on both Debian Linux and MacOs

## Installation
1. (For LinuX) Install SDL2 to system libraries (MacOs uses uncluded SDL2 library directories)
2. `make`

## Usage
`./RTv1 <scene_file>`
